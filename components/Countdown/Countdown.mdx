## API

Common props: `dataTheme`, `className`, `style`

### Countdown

| Property         | Description | Type      | Default |
| ---------------- | ----------- | --------- | ------- |
| box              | In boxes    | boolean   | false   |
| children         | -           | ReactNode | -       |
| value            | -           | number    | 0       |
| vertical         | -           | boolean   | false   |
| wrapperClassName | -           | string    | -       |

### Countdown.Group

| Property | Description | Type        | Default |
| -------- | ----------- | ----------- | ------- |
| box      | In boxes    | boolean     | false   |
| children | -           | Countdown[] | -       |
| vertical | -           | boolean     | false   |
