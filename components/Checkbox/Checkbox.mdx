## API

Common props: `dataTheme`, `className`, `style`

### Checkbox

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| checked | - | boolean | - |
| color | Toggle color | `primary` \| `secondary` \| `accent` \| `info` \| `success` \| `warning` \| `error` | - |
| defaultChecked | - | boolean | false |
| indeterminate | - | boolean | false |
| disabled | - | boolean | false |
| label | - | ReactNode | - |
| reverse | reverse label order | boolean | false |
| size | - | `xs` \| `sm` \| `md` \| `lg` | - |
| wrapperClassName | - | string | - |

It accepts all props which native `input[checkbox]` support.
