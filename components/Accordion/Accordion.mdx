## API

Common props: `dataTheme`, `className`, `style`

### Accordion

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| arrow | Adds arrow icon | `arrow` \| `plus` | - |
| bordered | - | boolean | true |
| children | - | ListOrItem\<ReactElement\> | - |
| defaultActive | - | number | 0 |
| join | join the items together | boolean | false |

### Accordion.Collapse

| Property | Description     | Type              | Default |
| -------- | --------------- | ----------------- | ------- |
| arrow    | Adds arrow icon | `arrow` \| `plus` | -       |
| bordered | -               | boolean           | true    |
| children | -               | ReactNode         | -       |
| open     | Force open      | boolean           | false   |

It accepts all props which [Collapse](/components/collapse/) support.
