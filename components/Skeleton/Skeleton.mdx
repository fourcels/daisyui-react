---
title: Skeleton
description: Skeleton is a component that can be used to show a loading state of a component.
---

## Examples

### Skeleton

```tsx live
<Skeleton className="w-32 h-32" />
```

### Skeleton - circle with content

```tsx live
<div className="flex flex-col gap-4 w-52">
  <div className="flex gap-4 items-center">
    <Skeleton circle className="w-16 h-16" />
    <div className="flex flex-col gap-4">
      <Skeleton className="h-4 w-20" />
      <Skeleton className="h-4 w-24" />
    </div>
  </div>
  <Skeleton className="h-32 w-full" />
</div>
```

### Skeleton - rectangle with content

```tsx live
<div className="flex flex-col gap-4 w-52">
  <Skeleton className="h-32 w-full" />
  <Skeleton className="h-4 w-28" />
  <Skeleton className="h-4 w-full" />
  <Skeleton className="h-4 w-full" />
</div>
```

## API

Common props: `dataTheme`, `className`, `style`

### Skeleton

| Property | Description | Type    | Default |
| -------- | ----------- | ------- | ------- |
| circle   | -           | boolean | false   |
