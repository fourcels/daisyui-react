---
title: Join
description: Join is a container for grouping multiple items, it can be used to group buttons, inputs, or any other element. Join applies border radius to the first and last item. Join can be used to create a horizontal or vertical list of items.
---

## Examples

### Join
<code src="./demo/basic.tsx" />

### Responsive
```tsx live
/**
 * description: it's vertical on small screen and horizontal on large screen
 */

<Join responsive>
    <Join.Item>
        <Button>Button</Button>
    </Join.Item>
    <Join.Item>
        <Button>Button</Button>
    </Join.Item>
    <Join.Item>
        <Button>Button</Button>
    </Join.Item>
</Join>

```

### Custom border radius
```tsx live
<Join>
    <Join.Item>
        <Input placeholder="Email"/>
    </Join.Item>
    <Join.Item>
        <Button className="rounded-r-full">Subscribe</Button>
    </Join.Item>
</Join>

```
### Join radio inputs with btn style
```tsx live
<Join>
    <Join.Item>
        <Button as="input" type="radio" name="options" aria-label="Radio 1"/>
    </Join.Item>
    <Join.Item>
        <Button as="input" type="radio" name="options" aria-label="Radio 2"/>
    </Join.Item>
    <Join.Item>
        <Button as="input" type="radio" name="options" aria-label="Radio 3"/>
    </Join.Item>
</Join>

```
