---
title: Join
description: Join is a container for grouping multiple items, it can be used to group buttons, inputs, or any other element. Join applies border radius to the first and last item. Join can be used to create a horizontal or vertical list of items.
---

## Examples

### Join

<code src="./demo/basic.tsx" />

### With extra elements in the group

```tsx live
<Join>
  <Join.Item>
    <Input placeholder="Search" />
  </Join.Item>
  <Join.Item>
    <Select
      placeholder="Filter"
      clearable={false}
      items={["Sci-fi", "Drama", "Action"]}
    />
  </Join.Item>
  <Indicator indicator={<Badge color="secondary">new</Badge>}>
    <Join.Item>
      <Button>Search</Button>
    </Join.Item>
  </Indicator>
</Join>
```

### Custom border radius

```tsx live
<Join>
  <Join.Item>
    <Input placeholder="Email" />
  </Join.Item>
  <Join.Item>
    <Button className="rounded-r-full">Subscribe</Button>
  </Join.Item>
</Join>
```

### Join radio inputs with btn style

```tsx live
<Join>
  <Join.Item>
    <Button as="input" type="radio" name="options" aria-label="Radio 1" />
  </Join.Item>
  <Join.Item>
    <Button as="input" type="radio" name="options" aria-label="Radio 2" />
  </Join.Item>
  <Join.Item>
    <Button as="input" type="radio" name="options" aria-label="Radio 3" />
  </Join.Item>
</Join>
```

## API

Common props: `dataTheme`, `className`, `style`

### Join

| Property   | Description | Type                       | Default |
| ---------- | ----------- | -------------------------- | ------- |
| children   | -           | ReactNode                  | -       |
| direction  | -           | `vertical` \| `horizontal` | -       |
| responsive | -           | boolean                    | false   |

### Join.Item

| Property | Description | Type         | Default |
| -------- | ----------- | ------------ | ------- |
| children | -           | ReactElement | -       |
