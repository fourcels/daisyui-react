### With a disabled button

```tsx live
<Pagination>
  <Pagination.Item>1</Pagination.Item>
  <Pagination.Item>2</Pagination.Item>
  <Pagination.Item disabled>...</Pagination.Item>
  <Pagination.Item>99</Pagination.Item>
  <Pagination.Item>100</Pagination.Item>
</Pagination>
```

### Extra small buttons

```tsx live
<Pagination>
  <Pagination.Item>«</Pagination.Item>
  <Pagination.Item>Page 22</Pagination.Item>
  <Pagination.Item>»</Pagination.Item>
</Pagination>
```

### Nex/Prev outline buttons with equal width

```tsx live
<Pagination className="grid grid-cols-2">
  <Pagination.Item variant="outline">Previous page</Pagination.Item>
  <Pagination.Item variant="outline">Next</Pagination.Item>
</Pagination>
```

### Using radio inputs

```tsx live
<Pagination name="pagination">
  <Pagination.Radio defaultChecked aria-label="1" />
  <Pagination.Radio aria-label="2" />
  <Pagination.Radio aria-label="3" />
  <Pagination.Radio aria-label="4" />
  <Pagination.Radio aria-label="5" />
</Pagination>
```

## API

Common props: `dataTheme`, `className`, `style`

### Pagination

| Property | Description | Type                                    | Default |
| -------- | ----------- | --------------------------------------- | ------- |
| children | -           | Pagination.Item[] \| Pagination.Radio[] | -       |
| name     | -           | string                                  | -       |
| size     | -           | `xs` \| `sm` \| `md` \| `lg`            | -       |

### Pagination.Item

It accepts all props which [Button](/components/button/) support.

### Pagination.Radio

It accepts all props which [Button\<"input"\>](/components/button/) support.
