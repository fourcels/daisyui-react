### Dialog modal with custom width

You can use any w-\* and max-w-\* utility class to customize the width

```tsx live
<Modal className="w-11/12 max-w-5xl" trigger={<Button>open modal</Button>}>
  <Modal.Header>Hello!</Modal.Header>
  <Modal.Body>Press ESC key or click the button below to close</Modal.Body>
</Modal>
```

### Responsive

Modal goes bottom on mobile screen and goes middle on desktop

```tsx live
<Modal responsive trigger={<Button>open modal</Button>}>
  <Modal.Header>Hello!</Modal.Header>
  <Modal.Body>Press ESC key or click the button below to close</Modal.Body>
</Modal>
```

## API

Common props: `dataTheme`, `className`, `style`

### Modal

| Property   | Description | Type                          | Default |
| ---------- | ----------- | ----------------------------- | ------- |
| backdrop   | -           | boolean                       | true    |
| children   | -           | ReactNode                     | -       |
| closable   | -           | boolean                       | true    |
| open       | Force open  | boolean                       | false   |
| position   | -           | `top` \| `middle` \| `bottom` | -       |
| responsive | -           | boolean                       | false   |
| trigger    | -           | ReactElement                  | -       |

### Modal.ActionButton

| Property | Description | Type                                | Default |
| -------- | ----------- | ----------------------------------- | ------- |
| onClick  | -           | (dialog: HTMLDialogElement) => void | -       |

It accepts all props which [Button](/components/button/) support.
