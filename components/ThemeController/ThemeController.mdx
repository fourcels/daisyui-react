## API

Common props: `dataTheme`, `className`, `style`

### ThemeController

| Property | Description | Type                       | Default |
| -------- | ----------- | -------------------------- | ------- |
| checked  | -           | boolean                    | false   |
| label    | -           | string                     | -       |
| labelAlt | -           | string                     | -       |
| onChange | -           | (checked: boolean) => void | -       |
| value    | -           | string                     | -       |

It accepts all props which [Toggle](/components/toggle/) support.
