## API

Common props: `dataTheme`, `className`, `style`

### Table

| Property | Description | Type                         | Default |
| -------- | ----------- | ---------------------------- | ------- |
| children | -           | ReactNode                    | -       |
| pinRows  | -           | boolean                      | false   |
| pinCols  | -           | boolean                      | false   |
| size     | -           | `xs` \| `sm` \| `md` \| `lg` | -       |
| zebra    | -           | boolean                      | false   |

### Table.Row

| Property | Description | Type      | Default |
| -------- | ----------- | --------- | ------- |
| active   | -           | boolean   | false   |
| children | -           | ReactNode | -       |
| hover    | -           | boolean   | false   |

### Table.Body

| Property | Description | Type      | Default |
| -------- | ----------- | --------- | ------- |
| children | -           | ReactNode | -       |
| hover    | -           | boolean   | false   |

### Table.Foot
