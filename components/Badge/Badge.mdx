### Badge in a button

```tsx live
<Button>
  Inbox <Badge>+99</Badge>
</Button>
<Button>
  Inbox <Badge color="secondary">+99</Badge>
</Button>
```

## API

Common props: `dataTheme`, `className`, `style`

### Badge

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| color | - | `neutral` \| `primary` \| `secondary` \| `accent` \| `info` \| `success` \| `warning` \| `error` \| `ghost` | - |
| outline | - | boolean | - |
| size | - | `xs` \| `sm` \| `md` \| `lg` | - |
