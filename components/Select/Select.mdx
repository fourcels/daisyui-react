## API

### SelectProps

Common props: `dataTheme`, `className`, `style`

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| color | Select color | `primary` \| `secondary` \| `accent` \| <br/> `info` \| `success` \| `warning` \| `error` \| <br/> `ghost` | - |
| bordered | Select with border | boolean | true |
| children | Select content | Option \| Option[] | - |
| disabled | Disabled state | boolean | false |
| items | Select items | SelectItem[] | - |
| placeholder | Select placeholder | SelectPlaceholder | - |
| size | Size of Select | `xs` \| `sm` \| `md` \| `lg` | - |

It accepts all props which native select support.

### Types
```tsx
type SelectItem = string | number | {
    label: string
    value?: string | number
}

type SelectPlaceholder = boolean | string | {
    content?: string
    disabled?: boolean = true
}

```
