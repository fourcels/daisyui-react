---
title: Browser mockup
description: Browser mockup shows a box that looks like a browser window.
---

## Examples

### Browser mockup

```tsx live
<MockupBrowser className="w-full" toolbar="https://daisyui.com">
  <div className="flex justify-center px-4 py-16 border-t border-base-300">
    Hello!
  </div>
</MockupBrowser>
```

### Browser mockup with background color

```tsx live
<MockupBrowser
  className="w-full border-base-content bg-base-300"
  toolbar="https://daisyui.com"
>
  <div className="flex justify-center px-4 py-16 bg-base-200">Hello!</div>
</MockupBrowser>
```

## API

Common props: `dataTheme`, `className`, `style`

### MockupBrowser

| Property | Description | Type      | Default |
| -------- | ----------- | --------- | ------- |
| children | -           | ReactNode | -       |
| toolbar  | -           | string    | -       |
