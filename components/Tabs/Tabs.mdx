### Tabs with custom color

```tsx live
<Tabs
  variant="lifted"
  activeClassName="[--tab-bg:yellow] [--tab-border-color:orange] text-primary"
>
  <Tabs.Tab label="tab 1">Tab 1</Tabs.Tab>
  <Tabs.Tab label="tab 2">Tab 2</Tabs.Tab>
  <Tabs.Tab label="tab 3">Tab 3</Tabs.Tab>
</Tabs>
```

## API

Common props: `dataTheme`, `className`, `style`

### Tabs

| Property        | Description | Type                              | Default |
| --------------- | ----------- | --------------------------------- | ------- |
| children        | -           | Tabs.Tab[]                        | -       |
| size            | -           | `xs` \| `sm` \| `md` \| `lg`      | -       |
| variant         | -           | `bordered` \| `lifted` \| `boxed` | -       |
| defaultActive   | -           | number                            | 0       |
| activeClassName | -           | string                            | -       |

### Tabs.Tab

| Property         | Description | Type      | Default |
| ---------------- | ----------- | --------- | ------- |
| active           | -           | boolean   | false   |
| activeClassName  | -           | string    | -       |
| children         | -           | ReactNode | -       |
| contentClassName | -           | string    | -       |
| disabled         | -           | boolean   | -       |
| indicator        | -           | ReactNode | -       |
| label            | -           | ReactNode | -       |
