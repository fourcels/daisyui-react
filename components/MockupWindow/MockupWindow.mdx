---
title: Window mockup
description: Window mockup shows a box that looks like an operating system window.
---

## Examples

```tsx live
<MockupWindow className="w-full">Hello!</MockupWindow>
```

## API

Common props: `dataTheme`, `className`, `style`

### MockupWindow

| Property         | Description | Type      | Default |
| ---------------- | ----------- | --------- | ------- |
| children         | -           | ReactNode | -       |
| contentClassName | -           | string    | -       |
