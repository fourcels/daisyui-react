## API

Common props: `dataTheme`, `className`, `style`

### Carousel

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| actions | Carousel with next/prev buttons | boolean | false |
| bleed | - | boolean | false |
| children | - | ListOrItem\<ReactElement\> | - |
| contentClassName | - | string | - |
| indicator | Carousel with indicator buttons | boolean | false |
| snap | Snap elements | `start` \| `center` \| `end` | - |
| vertical | Vertical carousel | boolean | false |
| width | - | `full` \| `half` | - |
| wrapperClassName | - | string | - |

### Carousel.Item

| Property | Description | Type             | Default |
| -------- | ----------- | ---------------- | ------- |
| children | -           | ReactNode        | -       |
| width    | -           | `full` \| `half` | -       |
