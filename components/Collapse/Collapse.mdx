## API

Common props: `dataTheme`, `className`, `style`

### Callapse

| Property         | Description     | Type              | Default |
| ---------------- | --------------- | ----------------- | ------- |
| arrow            | Adds arrow icon | `arrow` \| `plus` | -       |
| bordered         | -               | boolean           | true    |
| children         | -               | ReactNode         | -       |
| contentClassName | -               | string            | -       |
| open             | Force open      | boolean           | false   |
| title            | -               | ReactNode         | -       |
| titleClassName   | -               | string            | -       |
