## API

Common props: `dataTheme`, `className`, `style`

### Progress

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| color | - | `primary` \| `secondary` \| `accent` \| `info` \| `success` \| `warning` \| `error` | - |
| value | - | number | 0 |
| max | - | number | 100 |

It accepts all props which native `progress` support.
