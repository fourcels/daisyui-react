## API

Common props: `dataTheme`, `className`, `style`

### Loading

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| children | - | ReactNode | - |
| color | - | `neutral` \| `primary` \| `secondary` \| `accent` \| `info` \| `success` \| `warning` \| `error` | - |
| size | - | `xs` \| `sm` \| `md` \| `lg` | - |
| variant | - | `spinner` \| `dots` \| `ring` \| `ball` \| `bars` \| `infinity` | - |
