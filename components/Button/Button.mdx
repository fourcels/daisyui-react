### Buttons with different HTML tags

```tsx live

 * description: You can use `btn` class on <button>, <input>, <a>, etc...
 */

<Button type="submit">Submit</Button>
<Button type="reset">Reset</Button>
<Button as="a">Link</Button>
<Button as="input" type="button" value="Input" />
<Button as="input" type="submit" value="Submit" />
<Button as="input" type="radio" aria-label="Radio" />
<Button as="input" type="checkbox" aria-label="Checkbox" />
<Button as="input" type="reset" aria-label="Reset" />
```

## API

Common props: `dataTheme`, `className`, `style`

### Button

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| active | Force button to show active state | boolean | false |
| animation | Click animation | boolean | true |
| color | - | `neutral` \| `primary` \| `secondary` \| `accent` \| `info` \| `success` \| `warning` \| `error` \| `ghost` | - |
| children | - | ReactNode | - |
| disabled | Force button to show disabled state | boolean | false |
| endIcon | End Icon | ReactNode | - |
| block | Full width button | boolean | false |
| glass | Button with a glass effect | boolean | false |
| loading | Loading state | boolean | false |
| responsive | Different sizes on different viewpoints | boolean | false |
| shape | Shape of button | `circle` \| `square` | - |
| size | - | `xs` \| `sm` \| `md` \| `lg` | - |
| startIcon | Start Icon | ReactNode | - |
| as | Button with different HTML tags | `a` \| `button` \| `input` \| etc... | `button` |
| variant | Variant of button | `outline` \| `link` | - |
| wide | Wide button (more horizontal padding) | boolean | false |

It accepts all props which native `button` support.
