## API

Common props: `dataTheme`, `className`, `style`

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| active | 	Force button to show active state | boolean | false |
| animation | Click animation | boolean | true |
| color | Color of button | `neutral` \| `primary` \| `secondary` \| `accent` \| <br/> `info` \| `success` \| `warning` \| `error` \| <br/> `ghost` | - |
| children | Button content | ReactNode | - |
| disabled | Force button to show disabled state | boolean | false |
| endIcon | End Icon | ReactNode | - |
| fullWidth | Full width button | boolean | false |
| glass | Button with a glass effect | boolean | false |
| loading | Loading state | boolean | false |
| responsive | Button have different sizes <br/> on different browser viewpoints | boolean | false |
| shape | Shape of button | `circle` \| `square` |  |
| size | Size of button | `xs` \| `sm` \| `md` \| `lg` | `md` |
| startIcon | Start Icon | ReactNode | - |
| tag | Button with different HTML tags | `a` \| `button`  \| `input` \| etc... | `button` |
| variant | Variant of button | `outline` \| `link` | - |
| wide | Wide button (more horizontal padding) | boolean | false |

It accepts all props which native `button` support.
