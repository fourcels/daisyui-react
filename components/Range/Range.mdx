## API

Common props: `dataTheme`, `className`, `style`

### Range

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| color | Range color | `primary` \| `secondary` \| `accent` \| `info` \| `success` \| `warning` \| `error` | - |
| label | Show label | boolean \| (value: number) => ReactNode | false |
| max | - | number | 100 |
| min | - | number | 0 |
| measure | Show measure | boolean | false |
| size | - | `xs` \| `sm` \| `md` \| `lg` | - |
| step | - | number | 1 |
| wrapperClassName | - | string | - |

It accepts all props which native `input[range]` support.
